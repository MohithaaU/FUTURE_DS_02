Facebook Campaign Performance Analysis

Overview :
This project analyzes Facebook ad campaign performance using Power BI to understand how age and gender impact engagement, conversions, and cost efficiency.

Dataset:
-Source: Kaggle
-Includes impressions, clicks, ad spend, and conversions for demographic-based analysis.

Feature Engineering
  Additional KPIs created in Excel:
  -CTR,CPC,CPA
  -Conversion rate
  -ROI

Data Preparation

-Error checks and data type formatting

-ROI by age group
-CTR heatmap across demographics
-Spend vs conversions analysis
